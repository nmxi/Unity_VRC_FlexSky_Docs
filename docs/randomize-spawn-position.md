title: 空のバリエーションのランダム化
---

import ClickableImage from '@site/src/components/ClickableImage';

ワールドの雰囲気に合わせて、空のプリセットをランダムに切り替える設定を行えます。イベントや時間帯の演出に合わせて空の印象を軽く変えたい場合に便利です。

<ClickableImage src="/img/spawn-position-data-settings.png" alt="空のプリセット設定" />

- **Preset Mode（プリセットの選択方式）**
  - `Single` : 特定のプリセットを固定
  - `Random` : プリセット群からランダムに選択
- **Blend Mode（ブレンド方式）**
  - `Default` : 即時切り替え
  - `Align Scene` : シーンの演出タイミングに合わせて切り替え
  - `Random` : ランダムにブレンド

<ClickableImage src="/img/spawn-position-data-random-sample.png" alt="ランダム切り替えの範囲イメージ" />
<p style={{ textAlign: 'center' }}>Randomに設定されている時、シーンビューでプリセットの切り替え範囲を確認できます</p>
