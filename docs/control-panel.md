# コントロールパネルの追加

FlexSkyのコントロールパネルを使用すると、ワールド内で空のプリセットや明るさを切り替えられるようになります。

<ClickableImage src="/img/control-panel--hero.png" alt="FlexSkyコントロールパネル概要" />

## コントロールパネルの配置

<ClickableImage src="/img/add-FlexSky-control-panel.png" alt="コントロールパネルの配置" />

1. `Assets/mikinel/FlexSky/Prefabs/Additional` 内のコントロールパネルPrefab（同梱されている場合）をシーン上に配置
2. 位置の調整

## コントロールパネルの操作

### 空のプリセット切り替え
インスタンスに入室したときに空のプリセットを切り替えます。<br />イベントや時間帯に合わせて雰囲気を切り替えたい場合に便利です。
