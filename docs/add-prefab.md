# サンプルの配置

FlexSkyを導入する最初のステップは、サンプルPrefabまたはサンプルマテリアルをシーンに適用することです。最小限のセットアップだけで空の表現を切り替えられます。

## サンプルの場所

FlexSkyのアセット本体は以下の場所にあります：

```
Assets/mikinel/FlexSky
```

<ClickableImage src="/img/FlexSky-prefab-in-scene.png" alt="FlexSkyフォルダ内のサンプルの場所" />

## サンプルの適用

1. `Assets/mikinel/FlexSky` 内のサンプルPrefabまたはマテリアルを確認します。
2. サンプルPrefabをシーンに配置、またはマテリアルをSkyboxに割り当てます。
3. ワールド内で空の見え方を確認します。
