# About FlexSky

FlexSky is a shader for creating customizable skies for VRChat worlds.  
You can create high-quality daytime, nighttime, sunset, sunrise, and even surreal skies by adjusting parameters.

<ClickableImage src="/img/FlexSky_Art0.png" alt="FlexSky Preview" />

## Supported Environment

FlexSky is verified in the following environment:

- **Unity**: 2022.3.22f1
- **VRChat World SDK**: 3.10.1

## Resale and Redistribution

FlexSky code cannot be included in resale or redistribution.  
As an exception, preset data (with the .asset extension) can be sold or distributed.
